## ðŸ”§ Technical Tips

### File Size Optimization

**Keep Workbooks Under 10MB:**
- Extract data with only needed fields
- Aggregate in SQL before Tableau
- Remove unused data sources
- Hide unnecessary fields
- Limit data to relevant time period

**Why It Matters:**
- Faster load times
- Better user experience
- No upload issues

---

### Performance Best Practices

**For Smooth Interactivity:**
- Limit marks to < 10,000 per viz
- Use extracts, not live connections
- Simplify calculated fields
- Avoid complex LOD calculations
- Test filter performance

---

### Privacy & Data

**Tableau Public is PUBLIC:**
- All data is visible
- No password protection
- Cannot be made private

**Use:**
- Synthetic data
- Anonymized data
- Public datasets
- Aggregated data

**Never Upload:**
- Personal information (PII)
- Proprietary business data
- Confidential information
- Raw customer data
